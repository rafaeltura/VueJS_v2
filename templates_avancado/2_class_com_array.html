Class com ARRAY vue

Utilizando array para definir estilos visuais (css/class) no atributo class, por meio da diretiva V-BIND.

  • O Vue não permite o seguinte código :class="estilo1 estilo2 estilo3", visto que, cada 'estilo' representa uma chave no objeto vue. Caso, queira passar mais de um estilo para a mesma tag, então pode-se utilizar a dotação de array dentro do atributo class do html, :class="[estilo1, estilo2, estilo3]"
  • Tbm pode ser mesclada a aplicação da class. Exemplo, :class="[estilo1, { 'texto-branco': false, destaque: ehDestaque }]"
