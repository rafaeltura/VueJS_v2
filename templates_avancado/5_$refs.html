$REFS

Referenciando elementos HTML com $REFS

  • Representa uma propriedade do vue que nos permite referenciar elementos html e seus elementos filhos, é uma forma de pinçar esses elementos diretamente do DOM para alguma finalidade. É semelhante a seleção dos elementos html, que normalmente, é feita pelo js ou por meio do jquery.
  • Objeto, cuja as chaves, irão apontar para o elemento html que podemos selecionar. Para isso, deve-se add um atributo do próprio Vue ref no elemento html
  • Essa manipulação, por meio do $refs, não é reativa
