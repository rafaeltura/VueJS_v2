Class com objeto vue

Utilizando objetos para definir estilos visuais (css/class) no atributo class, por meio da diretiva V-BIND.

  • Pode-se criar um objeto diretamente no atributo class do html, onde a chave será o nome da classe css e o valor será um boolean, sendo true para aplicar e false para não aplicar o estilo.
  • Quando a classe css possui uma dotação a qual não pode ser aplicada a uma chave de objeto, então utiliza-se aspas no nome dessa chave.
  • É possível criar chaves no objeto data do vue para controlar cada estilo css, ou digitar diretamente o boolean para cada chave do objeto criado no atributo class do html.
