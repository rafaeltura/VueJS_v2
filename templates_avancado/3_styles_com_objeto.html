Style com objeto vue

Utilizando objetos para definir estilos visuais inline(style), por meio da diretiva V-BIND.

  • Pode-se criar um objeto diretamente no atributo style do html, onde a chave será o nome da propriedade css e o valor será de fato um valor, não um boolean como com a class.
  • Quando o style possui uma dotação a qual não pode ser aplicada a uma chave de objeto, então utiliza-se aspas no nome dessa chave ou o padrão camelCase.
  • É possível criar chaves no objeto data do vue para controlar cada style, ou digitar diretamente o valor para cada chave do objeto criado no atributo style do html.
